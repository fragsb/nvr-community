Checkpoint 1: Projeto NVR - Never CM
Linguagens de programação que pretendem usar:

Python (para automação e bots de integração)

JavaScript/React (para uma interface web de membros)

SQL (para gerenciamento do banco de dados da comunidade)

Resumo do projeto:

O projeto consiste no desenvolvimento de uma infraestrutura digital dedicada à comunidade de jogos NVR - Never CM, focando na integração entre membros e na organização de eventos competitivos e sociais. O objetivo principal é centralizar as informações da comunidade em uma plataforma única que facilite a comunicação e o engajamento dos jogadores.

Através de ferramentas automatizadas e um painel de controle administrativo, o sistema permitirá o gerenciamento de rankings internos, histórico de partidas e o cadastro de novos membros de forma simplificada. Isso visa fortalecer a identidade da marca NVR e proporcionar uma experiência mais profissional para os participantes.
